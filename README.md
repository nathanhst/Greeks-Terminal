# Greeks-Terminal
Options analytics dashboard build with Python, Dash and Plotly. Uses a fully vectorized BSM (Black-Scholes-Merton) engine to calculate and graph Greeks (delta, theta, gamma, vega, rho) in real-time. The vizualizer is based on the Jupyter Notebook also present in the repo and worked with Claude to prototype a dashboard quickly. It features simulated data generated locally (and provided in the repo) for AAPL options and generates simulated rolling expiration dates if it can't find a local AAPL options cache file. Feel free to launch it locally and try it out!
